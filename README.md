Мой вариант vindriktning-ikea основан на https://modkam.ru/2023/10/30/ikea-vindriktning/
с небольшими  доработками, убран родной контроллер, добавлено управление вентилятором и яркостью дисплея, измение цвета показаний взависимости от значений, индикация сонца над горизонтом, индикация ожидаемой погоды в зависимости от давления.
Все размешено на одной странице дисплея.
